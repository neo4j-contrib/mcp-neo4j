# Neo4j Memory MCP Server Quick Start

##  Installation, Testing

```bash
# 1. Build and run tests, clean
./scripts/test.sh build

# 2. Build Docker images 
./scripts/build.sh all

# 2. Publish Docker images 
./scripts/publish.sh
```
